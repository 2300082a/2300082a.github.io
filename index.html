How the CakeML compiler is verified and what does it mean to be verified(technical)

CakeML is a verified compiler. A compiler is a program that turns code written in a programming language to assembly code that runs on a computer. This translation is important to be correct as without that bugs may occur. Software verification is the process of proving using mathematics that a software specifies some specification. For a compiler the specification that people want to prove is semantics preservation. This means that the program behaves the same. Intuitively one may think of it similar to how 2 * 3 is the same as 2 + 2 + 2 as they both equal the same output, 6.
